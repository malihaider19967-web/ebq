HOW TO RESUME THIS CLAUDE CODE SESSION
=======================================

Run from the project root (/var/www/ebq):

  # Resume THIS exact conversation by its session id:
  claude --resume b9e7a986-13f4-408c-afbd-a145af6c5bc6

  # ...or pick it from an interactive list of past sessions:
  claude --resume          # then arrow to this one

  # ...or jump back into the most recent session in this directory:
  claude --continue

Full history is persisted on disk at:
  /root/.claude/projects/-var-www-ebq/b9e7a986-13f4-408c-afbd-a145af6c5bc6.jsonl


CAVEATS
-------
1. Resuming restores the CONVERSATION, not running processes. Crawls + the
   autoscaler keep running on their own (worker box + scheduler); re-check live
   state with: php artisan tinker, /admin/fleet, /horizon.
2. Background watchers I start in a shell die when the shell closes — harmless.
